# شركة رؤية المستقبل للخدمات الرقمية

### [رابط الموقع](https://nabilhadi.github.io/ruya-mustaqbal-site/)


موقع عرض تجريبي لصفحة هبوط عربية (RTL) صممت 
لعرض خدمات شركة رؤية المستقبل للخدمات الرقمية.

This repository contains a single-page Arabic landing site intended as a portfolio/demo project.

## ما الذي يحتويه المشروع
- `index.html` — الصفحة الرئيسية (RTL) مع الأقسام: هيرو، من نحن، خدمات، آراء العملاء، تواصل، وتذييل.
- `css/style.css` — الأنماط والتصاميم المتجاوبة، ظلال خفيفة، وتأثيرات التمرير.
- `js/script.js` — تفاعلية الواجهة: تمرير ناعم، رأس ثابت يتقلص، محاكاة إرسال نموذج، وأزرار واتساب.

## ميزات
- تصميم حديث ونظيف موجه للأعمال الصغيرة.
- دعم اللغة العربية وRTL باستخدام خط "Tajawal".
- تأثيرات دخول سلسة عند التمرير.
- بطاقات الخدمات مع تأثير hover.
- زر واتساب طافي مع إشعار توضيحي (رقم تجريبي).
- نموذج تواصل يعمل على الواجهة فقط (محاكاة إرسال).

## ملاحظات مهمة
- لا توجد معالجة خلفية — النموذج يعرض رسالة نجاح محاكاة فقط.
- رقم واتساب المستخدم هو عنصر تجريبي. لاستبداله افتح `js/script.js` وابحث عن المتغير `waNumber` وغيّره إلى رقمك بصيغة دولية.
- لتغيير رقم الهاتف الظاهر في الصفحة عدّل `index.html` في قسم الاتصال.

## معاينة وتشغيل محلي
أسهل طريقة لعرض الموقع هي فتح `index.html` في المتصفح مباشرة، أو تشغيل سيرفر بسيط لصفحات ثابتة.

باستخدام Python (متوافق مع Windows bash):
```bash
cd d:/programmingProjects/BusinessdDEMOWeb
python -m http.server 8000
# ثم افتح http://localhost:8000 في المتصفح
```

أو ببساطة افتح الملف `index.html` بالنقر المزدوج في مستعرض الملفات.

## تعديل سريع
- لتعديل نصوص العرض أو استبدال الصور/الرسومات، افتح `index.html` و`css/style.css` حسب الحاجة.
- لتحسين تجربة الإنتاج: استبدل رابط خط Google Fonts إذا رغبت في استضافة الخط محلياً.
